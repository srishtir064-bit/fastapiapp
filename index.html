FastAPI App

This project contains a FastAPI backend and a simple frontend folder structure.

Frontend files

  • frontend/app.js - application entry script
  • frontend/company.js - company helper module
  • frontend/test.js - sample test script
  • frontend/test1.js - sample utility script
  • frontend/package.json - frontend package metadata
  • frontend/package-lock.json - npm lockfile
  • frontend/node_modules/ - npm dependencies placeholder
  • frontend/talentspark/ - custom frontend folder placeholder

Backend endpoints

  • / - root status endpoint
  • /companies - company list and create
  • /jobs - job list and create
  • /about - about page
  • /contact - contact page
